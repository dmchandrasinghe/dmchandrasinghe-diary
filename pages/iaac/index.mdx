# 📦 Infrastructure as Code (IaaC)

Notes and patterns for managing infrastructure through code — repeatability, version control, and automation.

## Sub-topics

- **Terraform** – Providers, modules, state management, workspaces
- **Bicep** – Azure-native IaaC, modules, deployment stacks
- **ARM Templates** – Legacy Azure Resource Manager templates
- **Pulumi** – Infra with real programming languages
- **AWS CDK** – Cloud Development Kit for AWS
- **Ansible** – Configuration management and provisioning
- **Best Practices** – DRY principles, testing IaaC, drift detection
