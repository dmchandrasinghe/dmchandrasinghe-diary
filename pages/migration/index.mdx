# 🚀 Migration

Strategies, patterns, and real-world notes on migrating workloads, databases, and applications to the cloud.

## Sub-topics

- **Cloud Migration** – Assessment, planning, executing cloud migrations
- **Database Migration** – SQL, NoSQL migrations, Azure Database Migration Service
- **App Modernisation** – Containerising, refactoring legacy apps
- **Lift & Shift** – Rehosting on-prem workloads to cloud VMs
- **Replatforming** – Moving to managed services with minimal changes
- **Refactoring & Rearchitecting** – Cloud-native rewrites and re-architecting
- **Migration Tools** – Azure Migrate, Database Migration Service, AWS MGN
