# 🏗 Infrastructure

Cloud infrastructure design, platform engineering, and low-level system design notes.

## Sub-topics

- **Networking** – VNets, subnets, peering, VPN, ExpressRoute, firewall rules
- **Compute** – VMs, scale sets, container instances, serverless
- **Storage** – Blob, file shares, object storage, archival
- **DNS & Load Balancing** – Azure DNS, Traffic Manager, Application Gateway, Front Door
- **Security & IAM** – Identity, access, managed identities, Key Vault
- **Cloud Providers** – Azure, AWS, GCP specifics and comparisons
- **Platform Engineering** – Internal developer platforms, paved roads, self-service infra
